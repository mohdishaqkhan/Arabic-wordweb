# Arabic-Word-web
This website is going to be a goto for every arabic learner ,
as the existing online dictionaries and translations like al-maany provide so vague translations, a beginner can't 
understand it's proper use case `Arabic word-web` contains the entered word's:

1.    -root word
2.   3 synonyms
3.    3 antonyms
    3 example sentences(using the word in sentences with different use cases)
    the word's cultural origin and modern usages
